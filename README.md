# HousePricePrediction
# House Price Prediction using Linear Regression

## Prodigy InfoTech Machine Learning Internship - Task 1

### Objective
Build a machine learning model to predict house prices using Linear Regression.

### Dataset
House Prices Dataset (train.csv)

### Features Used
- GrLivArea (Ground Living Area)
- BedroomAbvGr (Number of Bedrooms)
- FullBath (Number of Full Bathrooms)

### Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

### Steps Performed
1. Loaded and explored the dataset.
2. Selected important features.
3. Split data into training and testing sets.
4. Trained a Linear Regression model.
5. Predicted house prices.
6. Evaluated model performance using:
   - MAE (Mean Absolute Error)
   - MSE (Mean Squared Error)
   - R² Score
7. Visualized Actual vs Predicted House Prices.

### Results
- Successfully predicted house prices using Linear Regression.
- Generated evaluation metrics and visualization graph.
